Add / Edit Connection Point

 

For adding a new connection point do the following steps:

  • Mark connection point as active layer in the layer list.
  • Activate the Edit Asset tool.
  • Start an edit session activating the Toggle Editing - tool.
  • Choose the Add Feature tool.
  • Click on the map to place the connection point.
  • After creating a new connection point the attribute dialog appears to enter relevant values.

For editing an existing connection point do the following steps:

  • Mark connection point as active layer in the layer list.
  • Activate the Edit Asset tool.
  • Start an edit session activating the Toggle Editing - tool.
  • Choose one of the following tools: Move Feature, Node Tool or Delete Selected .
  • Click on the connection point  you want to change. If you use the Delete Selected tool you have to select the connection point before.

For editing the attributes of an existing connection point do the following steps:

  • Mark connection point as active layer in the layer list.
  • Activate the Edit Asset tool.
  • Start an edit session activating the Toggle Editing - tool.
  • Click on the connection point you want to change. The attribute dialog appears and you can change the attributes.
