Add / Edit Fitting

 

For adding a new fitting do the following steps:

  • Mark fitting as active layer in the layer list.
  • Activate the Edit Asset tool.
  • Start an edit session activating the Toggle Editing - tool.
  • Choose the Add Feature tool.
  • Click on the map to place the fitting.
  • After creating a new fitting the attribute dialog appears to enter relevant values.

For editing an existing  fitting do the following steps:

  • Mark fitting as active layer in the layer list.
  • Activate the Edit Asset tool.
  • Start an edit session activating the Toggle Editing - tool.
  • Choose one of the following tools: Move Feature, Node Tool or Delete Selected .
  • Click on the fitting you want to change. If you use the Delete Selected tool you have to select the fitting before.

For editing the attributes of an existing fitting do the following steps:

  • Mark fitting as active layer in the layer list.
  • Activate the Edit Asset tool.
  • Start an edit session activating the Toggle Editing - tool.
  • Click on the fitting you want to change. The attribute dialog appears and you can change the attributes.
