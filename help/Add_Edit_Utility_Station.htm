Add / Edit Utility Station

 

For adding a new utility station do the following steps:

  • Mark utility station as active layer in the layer list.
  • Activate the Edit Asset tool.
  • Start an edit session activating the Toggle Editing - tool.
  • Choose the Add Feature tool.
  • Click on the map to place the utility station.
  • After creating a new utility station the attribute dialog appears to enter relevant values.

For editing an existing  utility station do the following steps:

  • Mark utility station as active layer in the layer list.
  • Activate the Edit Asset tool.
  • Start an edit session activating the Toggle Editing - tool.
  • Choose one of the following tools: Move Feature, Node Tool or Delete Selected .
  • Click on the utility station you want to change. If you use the Delete Selected tool you have to select the utility station before.

For editing the attributes of an existing utility station do the following steps:

  • Mark utility station as active layer in the layer list.
  • Activate the Edit Asset tool.
  • Start an edit session activating the Toggle Editing - tool.
  • Click on the utility station you want to change. The attribute dialog appears and you can change the attributes.
