Add / Edit Line Casing

 

For adding a new line casing do the following steps:

  • Mark line casing as active layer in the layer list.
  • Activate the Edit Asset tool.
  • Start an edit session activating the Toggle Editing - tool.
  • Choose the Add Feature tool.
  • Click on the map to place the line casing.
  • After creating a new line casing the attribute dialog appears to enter relevant values.

For editing an existing  line casing do the following steps:

  • Mark line casing as active layer in the layer list.
  • Activate the Edit Asset tool.
  • Start an edit session activating the Toggle Editing - tool.
  • Choose one of the following tools: Move Feature, Node Tool or Delete Selected .
  • Click on the line casing you want to change. If you use the Delete Selected tool you have to select the line casing before.

For editing the attributes of an existing line casing do the following steps:

  • Mark line casing as active layer in the layer list.
  • Activate the Edit Asset tool.
  • Start an edit session activating the Toggle Editing - tool.
  • Click on the line casing you want to change. The attribute dialog appears and you can change the attributes.
