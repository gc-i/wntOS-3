Add / Edit Maintenance



 

To add or edit a maintenance do the following steps:

  • Mark the appropriate layer as active layer in the layer list.
  • Activate the Edit Asset tool.
  • Start an edit session activating the Toggle Editing - tool.
  • Select the feature you want to add a new maintenance to or whose maintenances you want to edit
