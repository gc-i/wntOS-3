Add / Edit Flatrate Connection

 

For adding a new flatrate connection do the following steps:

  • Mark flatrate connection as active layer in the layer list.
  • Activate the Edit Asset tool.
  • Start an edit session activating the Toggle Editing - tool.
  • Choose the Add Feature tool.
  • Click on the map to place the flatrate connection.
  • After creating a new flatrate connection the attribute dialog appears to enter relevant values.

For editing an existing flatrate connection do the following steps:

  • Mark flatrate connection as active layer in the layer list.
  • Activate the Edit Asset tool.
  • Start an edit session activating the Toggle Editing - tool.
  • Choose one of the following tools: Move Feature, Node Tool or Delete Selected .
  • Click on the flatrate connection you want to change. If you use the Delete Selected tool you have to select the flatrate connection before.

For editing the attributes of an existing flatrate connection do the following steps:

  • Mark flatrate connection as active layer in the layer list.
  • Activate the Edit Asset tool.
  • Start an edit session activating the Toggle Editing - tool.
  • Click on the flatrate connection you want to change. The attribute dialog appears and you can change the attributes.
