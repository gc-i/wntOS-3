Add / Edit Damage



 

For adding a new damage do the following steps:

  • Mark damage as active layer in the layer list.
  • Activate the Edit Asset tool.
  • Start an edit session activating the Toggle Editing - tool.
  • Choose the Add Feature tool.
  • Click on the map to place the damage.
  • After creating a new damage the attribute dialog appears to enter relevant values.

    • To show complaint details at MAXCOM click the following icon:
  • Within  Set Damaged Asset all assets close to the new damage are listed. Select the relevant asset you want to assign the damage to and click Set:   

         

For editing an existing damage do the following steps:

  • Mark damage as active layer in the layer list.
  • Activate the Edit Asset tool.
  • Start an edit session activating the Toggle Editing - tool.
  • Choose one of the following tools: Move Feature, Node Tool or Delete Selected .
  • Click on the damage you want to change. If you use the Delete Selected tool you have to select the damage before.

For editing the attributes of an existing damage two approaches are possible:

First approach:

  • Mark damage as active layer in the layer list.
  • Activate the Edit Asset tool.
  • Start an edit session activating the Toggle Editing - tool.
  • Click on the damage you want to change. The attribute dialog appears and you can change the attributes (Figure 1).

Second approach:

  • Mark the appropriate layer as active layer in the layer list.
  • Activate the Edit Asset tool.
  • Start an edit session activating the Toggle Editing - tool.
  • Select the feature whose damages you want to edit. 
  • Choose the Damages  tab.
  • Select the damage you want to edit (Figure 2)
