Add / Edit Damage



To add or edit a damage do the following steps:

  • Mark the appropriate layer as active layer in the layer list.
  • Activate the Edit Asset tool.
  • Start an edit session activating the Toggle Editing - tool.
  • Select the feature you want to add a new damage to or whose damages you want to edit
