Add / Edit DMA

 

For adding a new DMA do the following steps:

  • Mark DMA as active layer in the layer list.
  • Activate the Edit Asset tool.
  • Start an edit session activating the Toggle Editing - tool.
  • Choose the Add Feature tool.
  • Click on the map to place the DMA.
  • After creating a new DMA the attribute dialog appears to enter relevant values.

For editing an existing DMA do the following steps:

  • Mark DMA as active layer in the layer list.
  • Activate the Edit Asset tool.
  • Start an editing session activating the Toggle Editing - tool.
  • Choose one of the following tools: Move Feature, Node Tool or Delete Selected .
  • Click on the DMA you want to change. If you use the Delete Selected tool you have to select the DMA before.

For editing the attributes of an existing DMA do the following steps:

  • Mark DMA as active layer in the layer list.
  • Activate the Edit Asset tool.
  • Start an edit session activating the Toggle Editing - tool.
  • Click on the DMA you want to change. The attribute dialog appears and you can change the attributes.
