Views



Purpose:

Most objects can be displayed in different views depending on the chosen attribute (e.g. pipeline type, pipeline material).

To display objects in different views do the following steps:

  • Mark the object whose view you want to change as active layer in the layer list.
  • Select the desired view. The map display and the legend change according to the view.

Or

  • Mark the object whose view you want to change as active layer in the layer list.
  • Choose one of the view icons in the WNT menu.
