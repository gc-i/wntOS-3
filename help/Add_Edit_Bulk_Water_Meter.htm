Add / Edit Bulk Water Meter

 

For adding a new bulk water meter do the following steps:

  • Mark bulk water meter as active layer in the layer list.
  • Activate the Edit Asset tool.
  • Start an edit session activating the Toggle Editing - tool.
  • Choose the Add Feature tool.
  • Click on the map to place the bulk water meter .
  • After creating a new bulk water meter the attribute dialog appears to enter relevant values.

For editing an existing bulk water meter do the following steps:

  • Mark bulk water meter as active layer in the layer list.
  • Activate the Edit Asset tool.
  • Start an edit session activating the Toggle Editing - tool.
  • Choose one of the following tools: Move Feature, Node Tool or Delete Selected .
  • Click on the bulk water meter  you want to change. If you use the Delete Selected tool you have to select the bulk water meter before.

For editing the attributes of an existing  bulk water meter do the following steps:

  • Mark bulk water meter as active layer in the layer list.
  • Activate the Edit Asset tool.
  • Start an edit session activating the Toggle Editing - tool.
  • Click on the bulk water meter you want to change. The attribute dialog appears and you can change the attributes.
