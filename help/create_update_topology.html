Build/Update Topology

Purpose:

After adding, editing or deleting assets network topology needs to updated. A correct and up-to-date network topology is required for any kind of topological analysis, such as network tracing. Please not that the WNT does not provide any tools to perform the actual topological analysis. However, it provides the Build/Update Topology tool to create proper topology.

To build or update network topology perform the following steps:

  • In the menu click the "Build/Update topology" icon:
  • In the opening window click Yes.
